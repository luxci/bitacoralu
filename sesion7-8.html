Sesión 7

En la clase de hoy, el profesor nos enseñó sobre dos conceptos fundamentales para la creación de un sitio web como el hosting y dominio. El hosting se refiere al servicio que proporciona un servidor para alojar los archivos y datos de un sitio web, permitiendo que este sea accesible en internet. Por otro lado, el dominio es la dirección única que identifica un sitio web en la red, fue una clase muy informativa y útil para comprender mejor cómo funciona la infraestructura básica de un sitio web.

Sesión 8

En la clase de hoy, el profesor nos enseñó sobre dos páginas web muy útiles para el desarrollo web: GitHub y Netlify. GitHub es una plataforma de alojamiento de código que permite a los desarrolladores colaborar en proyectos y controlar las versiones de su código. Netlify, por otro lado, es una plataforma de alojamiento web y automatización que facilita la implementación continua y la puesta en marcha de sitios web estáticos. Durante la clase, el profesor se mostraba muy entusiasmado tratando de ayudar a todos los alumnos a entender y utilizar estas herramientas, lo que hizo que la clase fuera muy dinámica y participativa.
